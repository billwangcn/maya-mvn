Laatste update: 2012-04-19

Dit zijn drie test scripts.
een receive kant (mayapy_receive : onze plug-in )
een verzend kant (mayapy_send : dient zogenaamd als MVN studio)
een verzend kant (mayapy_EULER_testmessage : deze kan gebruikt worden om 1 vol bericht van 668 bytes te testen)


OPZET:
- Download python 3.x vanaf de link die staat op onze interface pagina (of gewoon van Google)
- Open de IDLE (Python GUI), te vinden in je start menu of installatie folder
- Je hebt nu een python shell voor je, bij File-> Open-> Browse naar mayapy_receive
- Door op f5 te drukken run je deze script - deze loopt nu en wacht op een bericht
- Dubbelklik nu op mayapy (deze script wordt dan automatisch uitgevoerd door de python intepreter (python.exe))
- Je kunt zovaak drukken op mayapy als je wilt en het bericht kun je desgewenst aanpassen.
