Application: Maya 2012
Language: Python 3.x

startReading()


Parameters

Variables made ready by the 'lock()' command.

Return Values

Pose data which influences the locked human skeleton.

Description

Start reading motion capture ouput data from an external source (MVN Studio).
Input renders one 3d human skeleton its poses in real time.
This is loose reading, the motions are not saved.
In the current script external source is being listened from using udp port 5005.
